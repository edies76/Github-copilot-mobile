# Github Copilot Mobile

Esta es una aplicación móvil diseñada para llevar la potencia de GitHub Copilot a dispositivos móviles. Inspirada en Open Code, esta aplicación permite a los desarrolladores interactuar con el asistente de IA de GitHub directamente desde sus teléfonos o tablets.

## Características (Próximamente)

- **Asistente de Código en Tiempo Real**: Consulta sugerencias y resuelve dudas de programación.
- **Interfaz Móvil Optimizada**: Diseñada para ser fluida y fácil de usar en pantallas pequeñas.
- **Integración con GitHub**: Conexión directa con tu cuenta de GitHub.

## Desarrollo

Este proyecto está siendo desarrollado utilizando **Expo**, **React Native** y **TailwindCSS**.
